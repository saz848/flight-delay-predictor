FLIGHT DELAY PREDICTOR

Predicting Flight Cancellation and Delay using Flight Attributes and Weather Data

We want to be able to predict whether and how long a given flight will be delayed in departure, given the airline, route, and current weather conditions. This task is interesting and relevant because millions of people fly everyday, and generally would like for their flights to be on time. Having a tool that would be able to predict the length of delay for a given airline and route would be useful both when deciding on which flight to purchase and right before departure.
